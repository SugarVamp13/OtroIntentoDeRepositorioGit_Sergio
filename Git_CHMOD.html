Comando CHMOD

Este comando permite cambiar los permisos de archivos y directorios.

Permisos numéricos

Se utilizan números para definir permisos (lectura=4, escritura=2, ejecución=1).

chmod 755 archivo.sh

Permisos simbólicos

Permite añadir o quitar permisos usando letras.

chmod +x script.sh
