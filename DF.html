Comando DF

El comando DF muestra el espacio disponible y usado en los discos.

Sirve para controlar el almacenamiento del sistema.

Uso básico

df

Opciones comunes

-h: muestra tamaños en formato legible.

df -h

-T: muestra el tipo de sistema de archivos.

df -T

-a: muestra todos los sistemas de archivos.

df -a

Uso combinado

df -h /home
