Comando WC

El comando WC se utiliza para contar líneas, palabras y caracteres de un archivo.

Es muy útil para analizar contenido rápidamente.

Uso básico

wc archivo.txt

Opciones comunes

-l: cuenta líneas.

wc -l archivo.txt

-w: cuenta palabras.

wc -w archivo.txt

-c: cuenta caracteres.

wc -c archivo.txt

Uso combinado

cat archivo.txt | wc -l
