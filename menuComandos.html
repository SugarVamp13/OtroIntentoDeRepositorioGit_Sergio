Comandos de Linux

Hecho por: Juan Martin Prieto Ikot, Sergio Bellón Guzman, Hugo San Roman

FIND

Permite buscar archivos y directorios en el sistema.

PS

Muestra los procesos en ejecución.

GREP

Busca texto dentro de archivos usando patrones.

SORT

Ordena líneas de texto.

CUT

Extrae partes de cada línea.

HEAD

Muestra las primeras líneas de un archivo.

TAIL

Muestra las últimas líneas de un archivo.

CHMOD

Cambia permisos de archivos o directorios.

WC

Cuenta líneas, palabras y caracteres.

DF

Muestra el espacio disponible en disco.

DU

Muestra el uso de espacio.
