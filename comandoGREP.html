Comando GREP

El comando GREP busca texto dentro de archivos usando patrones.

Es muy utilizado para filtrar información en la terminal.

Uso básico

grep "texto" archivo.txt

Opciones comunes

-i: ignora mayúsculas y minúsculas.

grep -i "error" log.txt

-r: busca de forma recursiva en directorios.

grep -r "main" /home

-n: muestra el número de línea.

grep -n "texto" archivo.txt

-v: muestra las líneas que NO coinciden.

grep -v "error" log.txt

Uso combinado

ps aux | grep apache
