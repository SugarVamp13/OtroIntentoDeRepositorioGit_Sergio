Comando SORT

El comando SORT ordena líneas de texto de un archivo o entrada estándar.

Uso básico

sort archivo.txt

Opciones comunes

-r: ordena en orden inverso.

sort -r archivo.txt

-n: ordena numéricamente.

sort -n numeros.txt

-k: ordena por una columna específica.

sort -k 2 archivo.txt

-u: elimina duplicados.

sort -u archivo.txt

Uso combinado

cat archivo.txt | sort
